Install
-------
[安装依赖]

```bash
node >= 5.0.0
```
```bash
$ npm install -g typescript typings
$ typings install
$ npm install
```

Usage
-------
[用法]

#### development[开发环境]

```bash
$ npm start
```
访问：[http://127.0.0.1:8989](http://127.0.0.1:8989)

#### test[测试环境]

```bash
$ npm run test
```

#### production[生产环境]

```bash
$ npm run build
```
